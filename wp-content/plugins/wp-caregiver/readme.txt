Adds many options for tweaking frontend and backend of your WordPress site.

== Description ==

With this plugin you can turn on/off some theme and core services of WordPress or even you can customize it too.

= Frontend features =
* Remove generator meta tag
* Remove RSD (Really Simple Discovery ) link
* Remove Windows Live Writer link
* Remove post relational links (prev, next)
* Remove canonical link
* Remove RSS feed links
* Remove admin bar
* Remove the Website URL field from the comment form
* Remove allowed HTML tags text from the comment form
* Quick maintenance
* Force login
* Open Graph meta tag generator

= Backend features =
* Disable core (Wordpress) update notification
* Disable plugins update notification
* Disable themes update notification
* Hide login error messages
* Define custom login error message
* Enable "All Settings" item in Settings menu
* Enable Link Manager feature
* Enable Featured image support
* Enable excerpt field for pages
* Enable TinyMCE Editor for excerpt
* Enable categories for pages
* Display IDs in post/page list
* Customize WP Mail
* Remove Wordpress version from footer

= System information =
* Server time
* Memory usage
* SQL Queries
* Post revisions
* Disc usage

== Installation ==

1. Download the plugin from wordpress.
2. Extract all the files.
3. Upload everything (keeping the directory structure) to the /wp-content/plugins/ directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
